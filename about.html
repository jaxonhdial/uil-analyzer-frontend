About UIL Analyzer

UIL Analyzer is a searchable and filterable archive designed to make past UIL academic competition results more accessible and easier to explore.

UIL academic results have historically been fragmented across static archives and year-specific result pages with limited filtering options. UIL Analyzer centralizes this information into a single modern interface that allows competitors, coaches, and teams to explore historical data in meaningful ways.

UIL Analyzer is built by a competitor, for competitors, with a focus on clarity, accessibility, and long-term usability.

Current Version: v0.1

Core Features

Archives

The Archives page combines the benefits of the traditional UIL archive system, where results were accessible on a single page, with the recency and availability of SpeechWire results.

Users can filter results by year, event, conference, and competition level, making it easier to locate specific contests and placements.

Architecture

Frontend

  • Technologies: HTML, CSS, JavaScript
  • Role: User interface for searching and comparing results
  • Hosting: Netlify

Backend API

  • Technologies: Python, Flask
  • Role: Processes frontend queries and returns structured JSON
  • Hosting: Render

Data Query Layer

  • Technologies: Python, Pandas
  • Role: Queries the database and converts results into DataFrames

Data Population

  • Technologies: Python, Pandas, BeautifulSoup
  • Role: Scrapes official UIL archives to populate the database

Database

  • Technology: SQLite
  • Role: Primary data source storing all UIL competition results
